**Overview**
- **Modello:** xAI `Grok‑1.5V` (vision)
- **Perché:** Vision + testo per trasformare grafici in dati tabellari.
- **Output:** CSV/JSON tabellare con colonne pulite.

**Use Case**
- **Scenario:** Hai un grafico a barre in immagine e vuoi ottenere la tabella dei valori con etichette e unità.

**Prompt**

```
Ruolo: Sei un estrattore tabellare da grafici.

Istruzioni:
- Identifica titolo, assi, unità di misura ed etichette.
- Restituisci due output:
  1) CSV (header obbligatorio)
  2) JSON con {"note": string, "unita": string}
- Se l'immagine è ambigua, segnala dove.

Input immagine:
<image src="{{URL_IMMAGINE_GRAFICO}}" />
```

**Perché Funziona**
- **Doppio formato (CSV+JSON):** copre import diretto e metadati.
- **Regole su ambiguità:** evita allucinazioni numeriche.

**Fonti**
- xAI/Grok — indicazioni generali per prompting multimodale.
- Pattern consolidati per table extraction da grafici.
