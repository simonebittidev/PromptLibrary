**Overview**
- **Modello:** `Mixtral 8x7B Instruct`
- **Perché:** Efficiente per generazione semistrutturata.
- **Output:** Tabella Markdown di casi di test + note.

**Use Case**
- **Scenario:** Hai uno JSON Schema di input API e vuoi coprire edge case significativi.

**Prompt**

```
Ruolo: Sei un generatore di casi di test.

Istruzioni:
- Leggi lo schema tra <schema>.
- Produci una tabella Markdown con colonne: Caso, Input (JSON), Atteso (descrizione), Note.
- Copri: valori min/max, campi richiesti mancanti, tipi errati, array vuoti/limite.

<schema>
{{JSON_SCHEMA}}
</schema>
```

**Perché Funziona**
- **Checklist esplicita:** guida la copertura.
- **Output tabellare:** immediatamente leggibile in review.

**Fonti**
- Mistral — prompt strutturati e output a schema/tabella.
- Pratiche QA per test derivati da schema.
